refigure2 by rschroll

Plotting extension for Reinteract

refigure2 is a plotting extension for Reinteract that allows the use of the pylab syntax of matplotlib.

Dependencies

Reinteract and matplotlib

Install

Put the single .py file somewhere in your Python path
